Key design decisions
• Movie metadata never stored — always fetched from TMDB cache
• Anonymous users get a session_token cookie — same as signed-in UX
• Rooms auto-expire (24h) via expires_at
• Match check: query votes WHERE vote=true GROUP BY movie, check count = total members
• filters jsonb: {genres[], maxRuntime, minRating, mood}
