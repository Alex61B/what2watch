Core User Flow

The end-to-end journey from landing page to watching a movie together

🏠
Host
Creates a room
🔗
Guest
Opens shared link
↓
👤
Enter display name
Anonymous by default · optional sign-in to save preferences
↓
📺
Host: Select streaming services
Netflix · Prime · Hulu · HBO · Disney+ · Apple TV+
🎛️
Host: Set filters (optional)
Genre · Mood · Runtime · Min rating
Host-only setup steps
↓
⏳
Lobby — waiting for everyone to join
Host shares link · shows who's joined · host starts when ready
↓
⚙️
Server generates movie queue
Queries TMDB · filters by streaming services + filters · stores queue
↓
🗳️
Everyone votes — Tinder-style cards
Swipe Yes / No · see live "X of N have voted" progress · polling every 3s
↓
🎉
MATCH!
All voted Yes on same movie
Show match screen + Watch Now link
😅
No match
Queue exhausted
Adjust filters or start new session
